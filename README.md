The Simple Car Rental MVP is a dynamic web application providing a clean, user-friendly platform for browsing and renting vehicles. This project demonstrates frontend and backend fundamentals by building a reliable solution using a modern Java stack. The application connects to a database to store car details, user accounts, and rental records while utilizing visually appealing interfaces for the end user.
